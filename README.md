# App Description
-   Full stack application utilizing Amazon AWS for image storage.  
-   A Unsplash clone look-alike.  
-   Upload and view users images and profile.  

# Define Database Functionality + Input Data
1.  Redux stores User images as well as all site images.  
2.  User authentication provided by csrf and JWT token.  
3.  Upload connects to AWS bucket and redirect with public image displayed.  

# Tools
-   VS Code
-   Google Chrome Redux Dev tools
-   GitHub
-   AWS
-   Url references to AWS bucket stored in database

# Technologies
-   Deployment: Heroku
-   Frontend UI engine: React / Redux
-   Backend server: Express / Node JS
-   RDBMS: PostgresSQL v12
-   ORM: Sequelize
